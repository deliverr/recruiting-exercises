

InventoryAllocator.java	is class which has function which returns cheapest shipment in terms of list of hashmaps.

TestInventoryAllocator.java	 all the test class for InventoryAllocator.

ProjectTestSuiteRunner.java	Runner class for test Cases.

WareHouse.java  POJO class for providing sample Tests.




In order to Run:
Either include folder into IDE and Import Junit and RUN:


by Command Line:

Compile JAVA classes with:
javac ClassName.java

Run java class file with:
java ClassName

For tunning Junit Test Cases:
java -cp .:junit.jar org.junit.runner.JUnitCore ClassName
